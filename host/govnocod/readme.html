!!!Пояснения к коду!!!

 Данный код при загрузке страницы создаёт кнопку, которая при нажатии выводит время через 23 часа.
Основная функция "go" разбивает дату-время на символы и в цикле получает время, перегоняет его через функцию "trans", которая проверяет итерацию и в первой итерации перегоняет его через серию if-ов, получая время через 23 часа, а в остальных итерациях добавляет при необходимости слева ноль.
